# WinWin Telegram Bot with SofiaCash API

Telegram бот для управления депозитами и выплатами через API SofiaCash.

## 🚀 Функционал

### Для игроков:
- 💰 Пополнение счета (карты, ЮMoney, QIWI, крипто)
- 💸 Вывод средств
- 📊 Просмотр баланса и истории депозитов
- 🆘 Связь с поддержкой
- 📎 Загрузка чеков (PDF/фото)

### Для администраторов:
- 👥 Поиск игроков
- 📊 Статистика системы
- ⏳ Управление депозитами
- 📢 Рассылка сообщений
- 💼 Мониторинг баланса кассы
- ✅/❌ Принятие/отклонение депозитов
